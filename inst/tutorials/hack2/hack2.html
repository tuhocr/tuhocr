Cấu trúc file này như sau: crop_full.html

Trong file này thì cột item là tên các loại nông sản. Bạn hãy tìm cách trích xuất, làm sạch dữ liệu, sắp xếp làm sao đó để tách ra thành [danh sách các loại cây trồng] nhé.

Lưu ý là câu hỏi này hoàn toàn khác với câu hỏi từ Bài tập data cleaning cũng sử dụng cùng bộ dataset này.

Ý nghĩa của bài tập này là tình huống lọc dữ liệu ở dạng long (dữ liệu theo chiều dọc). Bạn cần trải dữ liệu ra dạng wide (dữ liệu theo chiều ngang) để thuận tiện trích xuất thông tin.

Làm rõ độ khó của bài tập

Đây là bài tập làm trên file dataset thực tế với dữ liệu lớn do đó bạn cần hình dung bản chất câu hỏi/logic bài tập này như thế nào trước khi tìm cách giải bài tập. Cụ thể là các loại nông sản ở cột item gồm những loại nông sản thuộc nhóm trồng trọt và chăn nuôi, với đặc trưng là tương ứng từng item sẽ có các element chứa thông tin liên quan.

Ví dụ item == "Rice" (thuộc nhóm trồng trọt) sẽ đi kèm với các element là biến production, area_harvested, và đôi khi là yield, còn item == "Chickens" (thuộc nhóm chăn nuôi) chỉ có element tương ứng là stocks.

Làm bài tập

Khi viết code online sẽ hơi bị chậm/lag. Bạn có thể thực hiện bài tập này offline theo hướng dẫn ở đây.

crop_full <- readRDS(file = "data/crop_production_all_data.rds")

Đáp án

Show/Hide

Có nhiều cách để giải bài tập này, ở đây mình tiếp cận theo cách dùng lệnh table() để lọc ra các tổ hợp giữa item và element sau đó reshape() qua dạng wide rồi lọc dữ liệu theo tính logic của câu chuyện như đã phân tích ở phần độ khó của bài tập này.

crop_full <- readRDS(file = "data/crop_production_all_data.rds")
ok <- as.data.frame(table(crop_full$item, crop_full$element))
ok_1 <- ok |> subset(Freq != 0)
ok_2 <- reshape(data = ok_1,
                idvar = c("Var1"),
                v.names = "Freq",
                timevar = "Var2",
                direction = "wide")
ok_2[!is.na(ok_2$Freq.area_harvested) & !is.na(ok_2$Freq.production), ] -> ok_3
as.character(ok_3$Var1) -> crop_item
crop_item
